Object Mode

Blender vključuje več različnih načinov za interakcijo s predmeti med katerimi sta glavno Object mode in Edit mode, ki ju lahko preklapljamo s tipko Tab. Object mode se uporablja za manipuliranje posameznih predmetov kot enote medtem, ko se Edit mode uporablja za manipuliranje podatkov predmeta. V Object mode lahko na primer premikamo, večamo, pomanjšujemo in obračamo cele poligonske mreže/polygon meshes.

Edit Mode

V Edit mode pa lahko na primer manipuliramo določena oglišča,robove in ploskve posamezne mreže.

Sculpt Mode

Način Sculpt Mode je podoben načinu Edit Mode, saj se uporablja za spreminjanje oblike modela, vendar se v načinu Sculpt Mode uporablja povsem drugačen potek dela: namesto posameznih elementov (vrhov, robov in obrazov) se območje modela spreminja s čopičem. Z drugimi besedami, namesto da bi izbral skupino vrhov, v načinu Sculpt Mode spreminja geometrijo v območju vpliva čopiča.

Vertex Paint

Vertex paint je preprost način barvanja predmeta z neposrednim spreminjanjem barve verteksov in ne tekstur. Ko je verteks pobarvan, se njegova barva spremeni glede na nastavitve čopiča. Barva vseh vidnih ploskev in robov, ki so povezani z verteksom, se nato spremeni z gradientom glede na barvo drugih povezanih vrhov. Barva zakritih obrazov se ne spremeni.

Weight Paint

Skupine verteksov imajo lahko zelo veliko število povezanih verteksov in s tem veliko število uteži (ena utež na dodeljen vrh). Weight paint je metoda za ohranjanje velikih količin informacij o teži na zelo intuitiven način. Uporablja se predvsem za oblikovanje mrež, kjer se skupine vrhov uporabljajo za določanje relativnih vplivov kosti na mrežo. Uporabljamo ga tudi za nadzor emisije delcev, gostote las, številnih modifikatorjev, tipk oblike itd. Vrhovom predmeta lahko dodelite uteži tako, da jih pobarvate s čopiči za uteži. Če začnete barvati mrežo, se uteži samodejno dodajo aktivni skupini vrhov (po potrebi se ustvari nova skupina vrhov).

Texture Paint

Blender ima vgrajen način barvanja, imenovan Texture Paint, ki je zasnovan posebej za hitro in enostavno urejanje UV-tekstur in slik v urejevalniku slik ali 3D-oglednem polju. Ker je tekstura UV samo slika za posebne namene, lahko uporabite tudi kateri koli zunanji program za barvanje, kot sta GIMP ali Krita. Ker ima lahko mreža plasti tekstur UV, lahko mrežo obarva več slik. Vendar ima vsaka tekstura UV samo eno sliko.
